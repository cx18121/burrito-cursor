## How to run

```bash
./scripts/build_app.sh
open ./BurritoCursor.app
```

First launch will:
1. Open the onboarding window automatically
2. Prompt for Camera permission (grant in System Settings → Privacy & Security → Camera)
3. Prompt for Accessibility permission when you click "Enable Cursor" (grant in System Settings → Privacy & Security → Accessibility)
4. Re-launch may be needed after granting Accessibility — macOS sometimes caches the prior state

## Acceptance checks

### Surface

- [ ] **Menu bar icon appears** on launch (hand emoji)
- [ ] **Onboarding window opens** on first run only
- [ ] **Onboarding shows live camera feed** with pose label updating in real time
- [ ] **Pose label correctly identifies** pointing / scrolling / indexBent / unknown

### Permissions

- [ ] **Camera permission prompt** appears on first onboarding open
- [ ] **Accessibility permission prompt** appears on first "Enable Cursor"
- [ ] **Denying permission** shows the explanatory alert (not a crash)
- [ ] **Revoking permission while ON** disables the app with the "Permission revoked" alert

### Cursor control

- [ ] **Pointing pose moves cursor** in the expected direction
- [ ] **No cursor drift during click** (visible jump < 5px when finger bends)
- [ ] **Index finger bend produces a single click** on the element under the cursor
- [ ] **Click does not get stuck** — verify mouseUp fires by clicking then dropping hand from view
- [ ] **Trackpad still works** while app is ON (additive coexistence)

### Scrolling

- [ ] **Two-finger pose scrolls** in a web page
- [ ] **Scroll direction matches expectation** (hand moves down → page scrolls down)

### Activation

- [ ] **Hotkey ⌃⌥H toggles** app on/off
- [ ] **Menu bar toggle** works (Enable Cursor / Disable Cursor)
- [ ] **Lid close while ON** triggers graceful disable (no stuck cursor on wake)
- [ ] **System sleep while ON** triggers graceful disable

### Adversarial

- [ ] **Multi-hand intrusion** — wave another hand into frame, cursor does not jerk to the new hand
- [ ] **Low-confidence frames** (poor lighting, hand near edge) — cursor freezes instead of jumping
